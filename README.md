Big-O Notation
O(n^2)



Time Complexity
Average Case: Aradığımız sayının ortada olması: O(n^2)
Worst Case: Aradığımız sayının sonda olması: O(n^2)
Best Case: Aradığımız sayının dizinin en başında olması: O(n)
